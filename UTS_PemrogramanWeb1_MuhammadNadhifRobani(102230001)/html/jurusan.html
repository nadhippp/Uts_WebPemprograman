S1 Teknik Informatika

Program Studi S1 Teknik Informatika menawarkan kurikulum yang komprehensif meliputi:

  • Pemrograman dan Pengembangan Software
  • Kecerdasan Buatan dan Machine Learning
  • Keamanan Siber
  • Pengembangan Aplikasi Mobile dan Web
  • Big Data dan Analitika

Lulusan kami siap untuk berkarir di berbagai sektor IT yang berkembang pesat.

S1 Teknologi Informasi

Program Studi S1 Teknologi Informasi berfokus pada:

  • Manajemen Sistem Informasi
  • Jaringan dan Infrastruktur IT
  • Cloud Computing
  • Internet of Things (IoT)
  • IT Governance dan Manajemen Proyek

Mahasiswa akan dipersiapkan untuk menjadi profesional IT yang mampu mengelola dan mengoptimalkan teknologi informasi dalam organisasi.

D3 Kriya Batik

Program Studi D3 Kriya Batik menawarkan pendidikan yang unik dalam seni batik, meliputi:

  • Sejarah dan Filosofi Batik
  • Teknik Pembuatan Batik Tradisional
  • Inovasi Desain Batik Modern
  • Manajemen Produksi Batik
  • Pemasaran Produk Batik

Lulusan akan memiliki keahlian untuk melestarikan dan mengembangkan seni batik Indonesia.
